# Python Practice Scripts

A collection of beginner-friendly Python scripts that cover core programming concepts such as variables, conditionals, loops, functions, OOP, file handling, and small utility games.

## 📁 Project Overview

This repository contains many standalone `.py` files, each focused on one topic or mini-problem, for example:

- Basic syntax and printing (`HelloWorld.py`, `input.py`)
- Conditionals and logic (`Conditional.py`, `PassOrFail.py`, `Grade.py`)
- Loops and patterns (`StarPattern.py`, `table.py`, `Naturalnum.py`)
- Functions and recursion (`Factorial.py`, `Recursive.py`, `Lambda.py`)
- OOP concepts (`Class_Object.py`, `Constructors.py`, `Inheritance.py`)
- File handling (`File_IO.py`, `copy_file.py`, `rename.py`)
- Mini projects (`Rock_Paper_Scissors.py`, `SecretCode.py`)

## ✅ Requirements

- Python 3.x installed

Check your Python version:

```bash
python3 --version
```

## ▶️ How to Run

Run any script individually from the project directory:

```bash
python3 filename.py
```

Example:

```bash
python3 HelloWorld.py
python3 Rock_Paper_Scissors.py
```

## 🧠 Learning Purpose

This repo is useful for:

- Python concept practice
- Quick syntax reference
- Beginner project ideas
- Interview/fundamentals revision

## 📌 Notes

- Most scripts are independent and do not depend on each other.
- Some scripts read/write files in the same directory (e.g., `.txt`, `.log`).
- A virtual environment folder (`myenv/`) is present.
